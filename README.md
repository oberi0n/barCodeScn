# Barcode Scanner

A static landing page that showcases the Barcode Scanner experience with a new logo.

## Preview
Open `index.html` in your browser. The page pulls in `assets/logo.svg` as the brand mark and
`styles.css` for styling.

## Assets
- `assets/logo.svg`: Brand logo with accessible metadata and alt text support.
- `styles.css`: Layout and component styles for the landing view.
